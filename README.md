##### Module-2 PC & Network Troubleshooting
## Introduction

PC and Network Troubleshooting involves identifying, diagnosing, and resolving hardware, software, and network-related issues. Understanding troubleshooting techniques helps maintain system performance, minimize downtime, and ensure reliable connectivity.

## 1. PC Hardware Troubleshooting

Hardware problems can occur due to faulty components, incorrect configurations, or physical damage.

Common Boot Issues
PC does not power on
No display on monitor
Continuous rebooting
Operating System not loading
Boot device not found
Solutions
Check power supply and cables.
Verify RAM installation.
Check storage connections.
Reset BIOS settings.
Repair boot files.

## 9. Networking Basics
# IP Address

Unique address assigned to every device.

Types
IPv4
IPv6
# Subnet Mask

Separates network and host portions of an IP address.

# Default Gateway

Allows communication outside the local network.

# MAC Address

Unique physical address of a network interface.

# DNS

Converts domain names into IP addresses.

# DHCP

Automatically assigns IP addresses to devices.
